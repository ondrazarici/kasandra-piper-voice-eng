# Kasandra ENG

More than a voice — a character designed for everyday conversations.

Kasandra ENG is a custom English female voice model for Piper TTS, created for Home Assistant, Wyoming Piper, and local AI assistant projects. The focus of this project is natural pacing, expressive intonation, and long-form speech quality rather than generic text-to-speech output.

## Features

- English female voice
- Optimized for Piper TTS

- Home Assistant ready
- Natural speech rhythm
- Expressive intonation
- Suitable for local AI assistants

## Installation

1. Copy `kasandra_eng.onnx`
2. Copy `kasandra_eng.onnx.json`
3. Place both files in your Piper/Wyoming Piper voice directory
4. Select the voice in your TTS configuration
